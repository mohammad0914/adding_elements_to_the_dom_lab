Text Duplicator

  • Make a form with id='text-duplicator-form'
  • The user should be able to enter a string into a text input with id='text-duplicator-string-input'
  • The user should be able to enter a number into a number input with id='text-duplicator-number-input'
  • They should then be able to click a "submit" button
  • Clicking the submit button will add that many copies of the string as list items to an unordered list with id='text-duplicator-list'
